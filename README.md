# 職場週年・抓周禮  
**Office Zhuazhou – 把社畜人生包裝成儀式**


小時候抓周：父母期待你抓書、抓筆、抓算盤  
長大後抓周：你抓到鍵盤、抓到咖啡杯、抓到主管的 K 條、抓到離職信……

這個專案把傳統抓周搬進職場，讓你（或你的同事）在螢幕前完成一場完整儀式：

- 輸入職場名號（e.g. 數據勇者 Alice）
- 選宇宙風格（物流職人 / 溫柔治癒 / 幽默金句 / 熱血冒險……）
- 披戰袍、吃福物（蘋果=清明、金雞腿=續航、敲鑼=一鳴驚人）
- 從米篩抓 3 件寶物（青蔥=聰明睿覺、芹菜=勤勞踏實、鏡子=明辨是非……）
- 踩兩個紅龜粿決定來年步局
- 產出個人化「抓周禮成證書」 → 長按截圖分享（或用來威脅同事）

適合場景：迎新、年終尾牙、部門破冰、離職派對、主管被整、自己一個人自癒……

**現在就試試 →** [https://oyc0813.github.io/career-gift/](https://oyc0813.github.io/career-gift/) 

## 快速開始

1. Clone 或直接開瀏覽器玩
2. 輸入名號 → 選風格 → 跟著儀式走
3. 抓完出證書 → 截圖發群組（建議配文：我的職場命盤，你們的呢？）

## 想客製 / 貢獻？

- 加新道具？修改 quotes？新增宇宙風格？
- 直接 PR 就好～最歡迎把你最酸的職場黑話加進去
- 也歡迎 issue 提「我公司專屬道具」想法（例如抓到「印章」= 永遠的 PM）
